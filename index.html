7) CUESTIONARIO

3. ¿Para qué se utiliza la transmisión asíncrona?
Se utiliza para describir el proceso en el que los datos transmitidos se codifican con bits de inicio y de detención, que especifican el principio y el final de cada carácter.
4. Menciona una ventaja/desventaja de la transmisión asíncrona:

5. Describe la diferencia entre Transmisión en Serie y Transmisión Paralela:
Una de las principales diferencias entre ambas es que en la transmisión en Serie, los datos se envían bit a bit mientras que en la Transmisión Paralela o en Paralelo se envía un byte (8 bits).
6. ¿Cómo trabaja la Transmisión en Serie?
Los datos se envían poco a poco de un ordenador a otro de forma bidireccional. Cada bit tiene su frecuencia de pulso de reloj.

9) CUESTIONARIO
1. ¿Cuál es la diferencia entre la ROM y la PROM?
La ROM se utiliza para la gestión del proceso de arranque, el chequeo inicial del sistema. La PROM es una memoria híbrida entre la RAM y la ROM. Con un software y hardware especial puedes grabar/borrar información dentro de ella (usualmente una cantidad limitada de veces) pero igual que la ROM, si la dejas sin energía, el programa o los datos que grabaste no se borrarán por ello.
2. ¿Cómo se borraran las memorias EPROM?
Una EPROM se puede borrar solamente mediante exposición a una fuerte luz ultravioleta.​
3. ¿Qué tipos de memorias existen?
Memoria RAM, Memoria ROM, Memoria PROM, Memoria EPROM, Memoria EEPROM
4. ¿Qué es el ferromagnetismo?
Es un fenómeno físico en el que se produce ordenamiento magnético de todos los momentos magnéticos de una muestra, en la misma dirección y sentido.
5. Menciona 3 ejemplos de almacenamiento ferromagnéticos:
Electroimán, Grabación de cinta magnetica, Transformadores.
